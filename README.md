# Saturn Leaderboard

A public, read-only Saturn points leaderboard dashboard.

## Public Data Sources

- Merkl Saturn point leaderboard
- Merkl Saturn point total
- Optional wallet lookup by public address

No wallet connection or signing is required.

## Files

- `index.html` - leaderboard dashboard
- `styles.css` - dashboard styling
- `app.js` - public data fetching and rendering
- `track-saturn-points.mjs` - optional command-line tracker
